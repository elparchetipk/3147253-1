Una recopilación de términos esenciales en el mundo de TI

A

Algoritmo

Conjunto ordenado y finito de operaciones que permite hallar la solución a un problema. En programación, es una secuencia de pasos lógicos que resuelve un problema específico.

API (Application Programming Interface)

Conjunto de reglas y especificaciones que las aplicaciones pueden seguir para comunicarse entre ellas, sirviendo de interfaz entre programas diferentes.

AWS (Amazon Web Services)

Plataforma de servicios de computación en la nube pública ofrecida por Amazon que incluye infraestructura, almacenamiento, bases de datos y herramientas de desarrollo.

B

Backend

Parte del desarrollo web que se encarga de toda la lógica de una página web que sucede en el servidor y bases de datos, no visible para el usuario.

Big Data

Conjunto de datos tan grandes y complejos que precisan de aplicaciones informáticas no tradicionales para tratarlos y analizarlos en tiempos razonables.

Blockchain

Estructura de datos que permite crear un registro digital de información y compartirlo entre una red de participantes independientes sin necesidad de una autoridad central.

Volver arriba

C

Cloud Computing

Modelo de servicio que permite acceder a recursos informáticos bajo demanda a través de internet, sin necesidad de invertir en infraestructura física.

CMS (Content Management System)

Software que facilita la creación, edición, organización y publicación de contenido digital, permitiendo a múltiples usuarios trabajar en el mismo entorno.

Ciberseguridad

Conjunto de tecnologías, procesos y prácticas diseñadas para proteger redes, dispositivos, programas y datos de ataques, daños o accesos no autorizados.

Volver arriba

D

DevOps

Conjunto de prácticas que combina el desarrollo de software (Dev) y las operaciones de TI (Ops) con el objetivo de acortar el ciclo de desarrollo y proporcionar una entrega continua de alta calidad.

Datos estructurados

Información que ha sido organizada en un formato específico, como bases de datos relacionales o hojas de cálculo, facilitando su procesamiento y análisis.

Volver arriba

E

Encriptación

Proceso de codificar información de manera que sólo las personas autorizadas puedan acceder a ella, utilizando algoritmos matemáticos para transformar datos legibles en datos cifrados.

ERP (Enterprise Resource Planning)

Software de gestión empresarial que integra y maneja los procesos operativos y de producción de una organización, como finanzas, recursos humanos, compras, ventas e inventario.

Volver arriba

F

Firewall

Sistema de seguridad informática que monitoriza y controla el tráfico de red entrante y saliente según reglas de seguridad predeterminadas, actuando como barrera entre redes seguras e inseguras.

Frontend

Parte del desarrollo web que se encarga de la interfaz visual con la que interactúan los usuarios, incluyendo diseño, navegación y experiencia de usuario.

Volver arriba

G

Git

Sistema de control de versiones distribuido diseñado para rastrear cambios en el código fuente durante el desarrollo de software, facilitando la colaboración entre programadores.

GPU (Graphics Processing Unit)

Unidad de procesamiento especializada en el cálculo de gráficos y operaciones de punto flotante, utilizada tanto para renderizar imágenes como para computación paralela en inteligencia artificial.

Volver arriba

H

HTML (HyperText Markup Language)

Lenguaje de marcado estándar para la creación de páginas web, que define la estructura y contenido básico de una página mediante etiquetas.

Hacking ético

Práctica de detectar vulnerabilidades en sistemas informáticos con el permiso explícito del propietario para mejorar la seguridad, en lugar de explotarlas maliciosamente.

Volver arriba

I

IoT (Internet of Things)

Red de objetos físicos ("cosas") que incorporan sensores, software y otras tecnologías con el fin de conectarse e intercambiar datos con otros dispositivos a través de internet.

IDE (Integrated Development Environment)

Aplicación informática que proporciona servicios integrales para facilitar el desarrollo de software, incluyendo editor de código, compilador, depurador y herramientas de automatización.

Volver arriba

J

JavaScript

Lenguaje de programación interpretado que permite implementar funcionalidades complejas en páginas web, como contenido actualizado dinámicamente, interfaces interactivas y animaciones.

JSON (JavaScript Object Notation)

Formato ligero de intercambio de datos, fácil de leer y escribir para humanos y fácil de analizar y generar para máquinas, basado en la sintaxis de objetos de JavaScript.

Volver arriba

K

Kubernetes

Sistema de código abierto para la automatización del despliegue, escalado y gestión de aplicaciones en contenedores, originalmente diseñado por Google y ahora mantenido por la Cloud Native Computing Foundation.

Kernel

Núcleo central de un sistema operativo que gestiona los recursos del sistema y facilita la comunicación entre el hardware y el software.

Volver arriba

L

Linux

Familia de sistemas operativos de código abierto basados en el kernel Linux, caracterizados por su estabilidad, seguridad y flexibilidad, ampliamente utilizados en servidores, supercomputadoras y dispositivos embebidos.

LAN (Local Area Network)

Red informática que conecta dispositivos en un área limitada como una casa, oficina o campus, permitiendo compartir recursos e información a alta velocidad.

Volver arriba

M

Machine Learning

Subcampo de la inteligencia artificial que proporciona a los sistemas la capacidad de aprender y mejorar automáticamente a partir de la experiencia sin ser programados explícitamente.

Microservicios

Enfoque arquitectónico para el desarrollo de software donde una aplicación se construye como un conjunto de pequeños servicios, cada uno ejecutándose en su propio proceso y comunicándose mediante mecanismos ligeros.

MySQL

Sistema de gestión de bases de datos relacional de código abierto, ampliamente utilizado para aplicaciones web y basado en el lenguaje SQL para manejar y consultar datos.
