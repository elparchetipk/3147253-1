N

NAS (Network Attached Storage)

Dispositivo de almacenamiento conectado a una red que permite almacenar y recuperar datos en un punto centralizado para usuarios autorizados y clientes heterogéneos de la red.

NoSQL

Tipo de sistema de gestión de bases de datos que no utiliza el modelo relacional y no requiere un esquema fijo, diseñado para manejar grandes volúmenes de datos distribuidos y no estructurados.

O

Open Source

Modelo de desarrollo de software basado en la colaboración abierta, donde el código fuente está disponible públicamente y puede ser modificado y distribuido por cualquier persona.

ORM (Object-Relational Mapping)

Técnica de programación que convierte datos entre sistemas de tipos incompatibles en lenguajes de programación orientados a objetos, creando una "base de datos virtual orientada a objetos".

Volver arriba

P

Python

Lenguaje de programación interpretado, de alto nivel y propósito general, enfocado en la legibilidad del código y utilizado ampliamente en análisis de datos, inteligencia artificial y desarrollo web.

PWA (Progressive Web App)

Tipo de aplicación web que utiliza tecnologías web modernas para proporcionar una experiencia similar a una aplicación nativa en dispositivos móviles, incluyendo funcionalidad sin conexión y notificaciones push.

Volver arriba

Q

QA (Quality Assurance)

Conjunto de actividades diseñadas para asegurar que los productos de software cumplan con los requisitos de calidad y funcionen según lo esperado, incluyendo pruebas, monitoreo y auditorías.

QR Code (Quick Response Code)

Código de barras bidimensional que puede ser leído por dispositivos móviles y almacena información en una matriz de puntos, utilizado para enlazar rápidamente a sitios web, realizar pagos o compartir contactos.

Volver arriba

R

REST API (Representational State Transfer)

Estilo arquitectónico para sistemas distribuidos que define un conjunto de restricciones utilizadas para crear servicios web escalables, permitiendo la comunicación entre sistemas mediante HTTP.

Router

Dispositivo de red que conecta múltiples redes y envía paquetes de datos entre ellas, determinando la mejor ruta para que los datos lleguen a su destino final.

Volver arriba

S

SaaS (Software as a Service)

Modelo de distribución de software donde las aplicaciones están alojadas en la nube y se accede a ellas a través de internet, generalmente mediante una suscripción, eliminando la necesidad de instalación y mantenimiento local.

SQL (Structured Query Language)

Lenguaje de programación específico diseñado para gestionar y manipular bases de datos relacionales, permitiendo realizar consultas, actualizaciones y administración de datos.

Scrum

Marco de trabajo ágil para la gestión y desarrollo de proyectos complejos, caracterizado por ciclos de trabajo cortos (sprints), reuniones diarias y roles específicos como Scrum Master y Product Owner.

Volver arriba

T

TCP/IP (Transmission Control Protocol/Internet Protocol)

Conjunto de protocolos de comunicación que definen cómo los datos deben ser formateados, transmitidos, enrutados y recibidos en internet, formando la base de la comunicación en línea.

Two-Factor Authentication (2FA)

Método de seguridad que requiere dos formas diferentes de identificación para acceder a algo, típicamente algo que el usuario conoce (contraseña) y algo que posee (como un teléfono móvil).

Volver arriba

U

UI (User Interface)

Espacio donde ocurren las interacciones entre humanos y máquinas, incluyendo los elementos visuales, la navegación y todos los componentes con los que los usuarios pueden interactuar.

UX (User Experience)

Conjunto de factores y elementos relativos a la interacción del usuario con un entorno específico, cuyo resultado es la generación de una percepción positiva o negativa de dicho servicio, producto o dispositivo.

Volver arriba

V

VPN (Virtual Private Network)

Tecnología que crea una conexión segura y encriptada sobre una red menos segura, como internet, permitiendo que los datos viajen de forma privada y protegiendo la información sensible.

Virtualización

Tecnología que permite crear versiones virtuales de recursos físicos como servidores, dispositivos de almacenamiento y redes, optimizando la utilización de hardware y mejorando la escalabilidad.

Volver arriba

W

Web 3.0

Evolución de la web caracterizada por la integración de tecnologías descentralizadas como blockchain, que busca devolver el control de los datos a los usuarios y reducir la dependencia de grandes plataformas centralizadas.

WebSocket

Protocolo de comunicación que proporciona canales de comunicación bidireccional y full-duplex sobre una única conexión TCP, permitiendo la interacción en tiempo real entre el navegador y el servidor.

Volver arriba

X

XML (eXtensible Markup Language)

Lenguaje de marcado que define reglas para codificar documentos en un formato legible tanto para humanos como para máquinas, utilizado para estructurar, almacenar e intercambiar datos.

XSS (Cross-Site Scripting)

Tipo de vulnerabilidad de seguridad informática que permite a un atacante inyectar scripts del lado del cliente en páginas web vistas por otros usuarios, pudiendo robar información o realizar acciones no autorizadas.

Volver arriba

Y

YAML (YAML Ain't Markup Language)

Formato de serialización de datos legible por humanos, utilizado principalmente para archivos de configuración y en aplicaciones donde los datos son almacenados o transmitidos.

Volver arriba

Z

Zero-day Vulnerability

Vulnerabilidad de seguridad informática desconocida para el desarrollador del software, que puede ser explotada por atacantes antes de que se cree y publique un parche para resolverla.

ZIP

Formato de archivo comprimido que permite reducir el tamaño de los archivos para facilitar su almacenamiento y transferencia, soportando varios métodos de compresión y encriptación.
